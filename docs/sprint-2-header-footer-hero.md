# Sprint 2 Header Footer Hero

## Objective

Implement the first visible brand layer without building the full TOP page.

## Implemented

- Sticky header with logo placeholder and X-PROSPECT wordmark
- Center desktop navigation
- Mobile hamburger menu with `aria-expanded`, `aria-controls`, Escape close, and link-close behavior
- Scroll state for white background and subtle shadow
- Hero section with slogan, supporting copy, CTA buttons, and abstract visual placeholder
- Footer with brand, slogan, navigation, placeholder company information, contact/privacy links, and copyright
- Front-page template with placeholder section titles only

## Front Page Sections

- Problems
- Services
- Before After
- Why X-PROSPECT
- Technology
- Contact

## Validation

- `theme.json` parse check
- JavaScript syntax check with bundled Node.js
- PHP lint when PHP is available
